class cryptocurrency_transactions.CryptoTransaction(crypto_type, amount, date)¶

Базовые классы: object

Представляет транзакцию криптовалюты.

crypto_type¶

Тип криптовалюты (например, „Bitcoin“, „Ethereum“).

Type:

str

amount¶

Сумма транзакции.

Type:

float

date¶

Дата транзакции в формате „YYYY-MM-DD“.

Type:

str

display_transaction()¶

Выводит информацию о транзакции в формате строки.

Пример:
>>> transaction = CryptoTransaction('Bitcoin', 0.5, '2023-10-01')
>>> transaction.display_transaction()
Crypto: Bitcoin, Amount: 0.5, Date: 2023-10-01
cryptocurrency_transactions.calculate_tax(amount)¶

Вычисляет налог на сумму транзакции.

Параметры:

amount (float) – Сумма, на которую будет рассчитан налог.

Результат:

Сумма налога, равная 20% от указанной суммы.

Тип результата:

float

Исключение:

ValueError – Если amount отрицательное.

Пример:
>>> calculate_tax(100)
20.0
class financial_automation.AutomatedFinanceSystem(user)¶

Базовые классы: object

Автоматизированная финансовая система для управления пользователем.

user¶

Пользователь, для которого выполняются операции.

Type:

User

report¶

Финансовый отчет пользователя.

Type:

FinancialReport

Калькулятор налога на доход пользователя.

Type:

TaxCalculator

automate_operations()¶

Автоматически выполняет несколько операций: пополнение, перевод, снятие.

В процессе выполнения операций: - Баланс пользователя пополняется случайной суммой. - Снимается случайная сумма, если это возможно. - Может быть выполнен перевод средств другому пользователю.
