Документация модуля Task Management

Модуль Task Management отвечает за управление задачами, включая создание, отслеживание и управление проектами. Он предоставляет функциональность для работы с шаблонами задач, командной работы и управления майлстоунами.

Классы и функции модуля

1. Класс Task

Представляет задачу в системе.

Атрибуты:

  • title (str): Название задачи.
  • description (str): Описание задачи.
  • due_date (str): Срок выполнения задачи в формате 'YYYY-MM-DD'.
  • status (str): Статус задачи (например, 'выполнена', 'в процессе').

Методы:

  • __init__(self, title, description, due_date): Инициализирует объект Task.
  • mark_as_completed(self): Устанавливает статус задачи как 'выполнена'.
  • update_status(self, status): Обновляет статус задачи.

2. Класс TaskTemplate

Представляет шаблон задачи для упрощения создания типовых задач.

Атрибуты:

  • template_name (str): Название шаблона.
  • default_description (str): Описание по умолчанию для задач, созданных из шаблона.

Методы:

  • __init__(self, template_name, default_description): Инициализирует объект TaskTemplate.
  • create_task(self, title, due_date): Создает задачу на основе шаблона.

3. Класс Milestone

Представляет майлстоун в проекте.

Атрибуты:

  • milestone_name (str): Название майлстоуна.
  • due_date (str): Срок выполнения майлстоуна в формате 'YYYY-MM-DD'.
  • tasks (list): Список задач, связанных с майлстоуном.
