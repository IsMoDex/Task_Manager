Документация модуля Financial Operations

Модуль Financial Operations отвечает за управление финансовыми транзакциями пользователей, включая операции пополнения, снятия и перевода средств. Он также предоставляет функциональность для ведения финансовых отчетов и расчета налогов.

Классы и функции модуля

1. Класс User

Представляет пользователя с балансом.

Атрибуты:

  • username (str): Имя пользователя.
  • balance (float): Баланс пользователя.

Методы:

  • __init__(self, username, balance=0): Инициализирует объект User.
  • deposit(self, amount): Пополнение баланса пользователя.
  • withdraw(self, amount): Снятие средств с баланса.
  • transfer(self, amount, recipient): Перевод средств между пользователями.

2. Класс FinancialReport

Представляет финансовый отчет для пользователя.

Атрибуты:

  • user (User): Пользователь, для которого создается отчет.
  • transactions (list): Список транзакций пользователя.

Методы:

  • __init__(self, user): Инициализирует объект FinancialReport.
  • record_transaction(self, transaction_type, amount): Записывает финансовую операцию в журнал.
  • generate_report(self): Генерирует отчет о финансовых операциях пользователя.

3. Класс TaxCalculator

Рассчитывает налог на доход.

Атрибуты:

  • income (float): Доход пользователя.
  • tax_rate (float): Ставка налога (по умолчанию 15%).

Методы:

  • __init__(self, income, tax_rate=0.15): Инициализирует объект TaxCalculator.
  • calculate_tax(self): Рассчитывает налог на доход.
